主要特性：
自动登录: 自动处理用户名密码登录
案件列表获取: 从案件综合查询页面获取所有案件
详情页爬取: 获取每个案件的完整信息
数据分类保存: 按案件编号创建文件夹，分别保存各类信息
断点续传: 支持中断后继续爬取
错误处理: 完善的异常处理和重试机制

使用方法：
安装依赖：pip install requests beautifulsoup4
运行程序：python ZMJGCaseScraper.py
输入用户名和密码
选择基础版本或支持断点续传的版本
等待爬取完成
程序会自动处理登录、获取案件列表、爬取详情并按案件编号分别保存所有信息。
